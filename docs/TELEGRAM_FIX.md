# Помилка «Telegram не налаштовано» — виправлення за 2 хв

Сайт працює. Google Apps Script **ще не бачить токен бота**.

Перевірка: відкрийте  
`https://script.google.com/macros/s/AKfycbzV-97Hoa0YdyK_DZBdgIkNHe1fKqmfn6To62xyFQ4P9oZq1wAg6sVL7LUUgrmH0Pz-UA/exec?action=ping`

Поки `"telegram":false` — замовлення з кошика не підуть.

---

## Спосіб 1 — ▶ Run (найпростіше)

1. Google **Таблиця** → **Розширення** → **Apps Script**.
2. Відкрийте файл з кодом (як у `scripts/google-apps-script.gs` у GitHub — **вставте останню версію**, якщо давно не оновлювали).
3. Знайдіть функцію **`setupTelegramProperties`**.
4. Замініть рядок:
   ```javascript
   var TOKEN = "ВСТАВТЕ_ТОКЕН_ВІД_BOTFATHER_СЮДИ";
   ```
   на (приклад):
   ```javascript
   var TOKEN = "8809482654:AAH...ваш_повний_токен...";
   ```
5. У випадаючому списку функцій виберіть **`setupTelegramProperties`**.
6. Натисніть **▶ Виконати** (Run). Дозвольте доступ, якщо запитає.
7. Унизу **Журнал виконання** — без помилки.
8. **Видаліть токен** з рядка `TOKEN` (залиште знову заглушку) → **Зберегти** (Ctrl+S).

Знову відкрийте `?action=ping` — має бути **`"telegram":true`**.

---

## Спосіб 2 — Script Properties

1. Apps Script → **⚙️ Налаштування проєкту** (зліва внизу).
2. **Властивості скрипта** → **Додати властивість скрипта**:

| Property | Value |
|----------|--------|
| `TELEGRAM_BOT_TOKEN` | повний токен від @BotFather |
| `TELEGRAM_CHAT_ID` | `-1003933471474` |

3. Зберегти.

## Розгортання веб-додатку (важливо)

**Розгорнути** → **Керувати розгортаннями** → олівець → переконайтесь:

- **Виконувати від імені:** **Я** (не «користувач, який відкриває»).
- **Хто має доступ:** **Усі**.

Якщо змінювали код — **Нова версія**.
